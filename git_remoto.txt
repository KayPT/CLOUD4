Comandos git

git clone - Usado para 'clonar' repositórios/projectos remotos já existentes no Github.
Uso: "git clone {url-repositório}"

git pull - Usado para copiar o conteúdo de um 'branch' de um repositório remoto e sobrepõe todas as alterações no 'branch' atual.
Uso: "git pull {remoto}"

git push - Usado para enviar todo o conteúdo do repositório local para o Github
Uso: "git push {nome-remoto}"
